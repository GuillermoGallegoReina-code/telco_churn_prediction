customerID: Identificador único del cliente.
gender: Género del cliente (Masculino/Femenino).
SeniorCitizen: Indica si el cliente es una persona mayor.
Partner: Estado civil (Tiene pareja: Sí/No).
Dependents: Indica si el cliente tiene dependientes.
tenure: Meses que el cliente ha estado con la empresa.
PhoneService: Si el cliente tiene servicio telefónico.
MultipleLines: Si el cliente tiene múltiples líneas.
InternetService: Tipo de servicio de internet (DSL, Fibra óptica, Ninguno).
OnlineSecurity, OnlineBackup, DeviceProtection, TechSupport, StreamingTV, StreamingMovies: Servicios adicionales contratados (Sí/No).
Contract: Tipo de contrato (Mensual, Un año, Dos años).
PaperlessBilling: Si el cliente tiene facturación electrónica.
PaymentMethod: Método de pago (Débito automático, Transferencia bancaria, etc.).
MonthlyCharges: Cargos mensuales.
TotalCharges: Cargos totales.
Churn: Indica si el cliente ha abandonado el servicio (Sí/No).
